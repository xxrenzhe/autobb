# 用户手册 (User Guide)

## 欢迎使用AutoAds！

AutoAds是一款强大的Google Ads自动化工具，帮助您快速创建和管理Google搜索广告。本手册将指导您完成从注册到成功上线广告的全流程。

```
Step 1: 连接Google Ads账号 (2分钟)
   ↓
```

## 一、连接Google Ads账号

### 1.1 为什么需要连接账号？

AutoAds需要访问您的Google Ads账号才能帮您创建和管理广告。我们使用Google官方OAuth认证，安全可靠。

### 1.2 操作步骤

#### Step 1: 点击"连接Google Ads账号"

1. 访问AutoAds首页
2. 点击页面中央的 **"连接Google Ads账号"** 按钮
3. 系统将跳转到Google授权页面

![连接账号按钮](./images/connect-button.png)

#### Step 2: 登录Google账号

1. 使用您的Google Ads账号登录
2. 如果已登录，会直接显示授权页面

#### Step 3: 授权AutoAds访问

1. 阅读权限说明：
   - 查看和管理您的Google Ads广告系列
   - 查看广告表现数据
2. 点击 **"允许"** 授权

#### Step 4: 选择要管理的Google Ads账号

1. 系统会显示您可访问的所有Google Ads账号
2. 选择您要管理的账号（支持多选）
3. 点击 **"确认选择"**

![选择账号](./images/select-account.png)

#### Step 5: 完成连接

- 系统显示 "✅ 账号连接成功"
- 自动跳转到Dashboard页面

### 1.3 数据安全说明

**您的数据存储在哪里？**
- ✅ 所有数据存储在**您的浏览器本地**（IndexedDB）
- ✅ AutoAds **不会上传您的数据到服务器**
- ✅ Google Ads访问令牌使用AES-256加密存储

**注意事项**：
- ⚠️ 清除浏览器数据会导致本地数据丢失
- ⚠️ 更换浏览器或设备需要重新连接账号
- ✅ 建议定期使用"数据导出"功能备份数据

## 二、创建营销主张Offer

### 2.1 什么是Offer？

Offer（营销主张）是您要推广的产品或服务的核心信息，包括：
- 品牌名称
- 产品描述
- 落地页URL
- 目标关键词

一个Offer可以创建多个广告系列（支持A/B测试）。

### 2.2 创建Offer

#### Step 1: 进入Offer列表页

- 点击左侧导航栏的 **"营销主张"**
- 或者从Dashboard点击 **"创建首个Offer"**

#### Step 2: 点击"新建Offer"

点击右上角的 **"+ 新建Offer"** 按钮

#### Step 3: 填写Offer信息

**必填字段**：

1. **Offer名称**（内部使用）
   ```
   示例：Nike跑步鞋秋季促销
   ```
   提示：方便您识别，不会出现在广告中

2. **品牌名称**
   ```
   示例：Nike
   ```
   提示：会出现在广告标题中

3. **产品描述**（100-300字）
   ```
   示例：Nike最新款专业跑步鞋，透气网面设计，Air Zoom气垫缓震，
   适合长跑和马拉松训练。多种配色可选，现在下单享8折优惠。
   ```
   提示：描述越详细，AI生成的广告创意质量越高

4. **推广链接 (Promotion URL)**（新增）
   ```
   示例：https://amzn.to/abc123
   ```
   提示：
   - 填写联盟追踪链接或带参数的推广链接
   - 此链接会自动设为Landing Page URL
   - 确保用户点击广告后访问带追踪的链接

5. **推广国家 (Target Country)**（新增）
   ```
   选项：美国(US)、英国(UK)、德国(GE)、法国(FR)、西班牙(ES)、
        意大利(IT)、日本(JP)、中国(CN)等
   ```
   提示：
   - 选择广告投放的目标国家
   - 影响AI生成的语言和广告定位
   - 选择后系统自动推导推广语言

   ⚠️ **重要限制**：
   - **一个Offer只支持单一国家/语言推广**
   - 如需多语言推广同一产品，需为每种语言创建独立的Offer
   - 示例：推广美国创建"产品-US"，推广德国创建"产品-DE"

6. **推广语言 (Target Language)**（自动推导，只读）
   ```
   示例：选择"美国(US)" → 自动显示"English"
        选择"德国(GE)" → 自动显示"German"
        选择"法国(FR)" → 自动显示"French"
   ```
   提示：
   - 此字段不可手动编辑
   - 根据推广国家自动设置
   - AI会使用此语言生成广告文案

   **多语言推广场景示例**：
   ```
   场景：您想在美国、德国、法国推广同一款产品

   正确做法：创建3个独立的Offer
   ✅ Offer 1: "Nike跑鞋-美国"
      - 推广国家：美国(US)
      - 推广语言：English（自动）
      - 关键词：Nike running shoes, sports shoes...

   ✅ Offer 2: "Nike跑鞋-德国"
      - 推广国家：德国(GE)
      - 推广语言：German（自动）
      - 关键词：Nike Laufschuhe, Sportschuhe...

   ✅ Offer 3: "Nike跑鞋-法国"
      - 推广国家：法国(FR)
      - 推广语言：French（自动）
      - 关键词：Chaussures de course Nike...

   错误做法：
   ❌ 一个Offer同时选择US、GE、FR（系统不支持）

   好处：
   - 每个国家可以使用不同的关键词和广告文案
   - 便于分别追踪各国广告表现
   - 灵活调整各国预算和策略
   ```

7. **店铺/详情页 URL (Product Page URL)**（新增）
   ```
   示例：https://www.amazon.com/nike-running-shoes/dp/B08XXX
   ```
   提示：
   - 填写商品详情页或店铺页面URL
   - 用于AI抓取商品信息和详情
   - 与推广链接不同：此URL用于读取信息，不带追踪参数

8. **Landing Page URL**（自动生成，只读）
   ```
   自动等于推广链接，无需手动填写
   ```
   提示：
   - 此字段不可手动编辑
   - 自动等于推广链接(Promotion URL)
   - 确保广告点击后访问带追踪的链接

9. **目标关键词**（5-20个）
   ```
   示例：
   - Nike跑步鞋
   - 专业跑鞋
   - 马拉松跑鞋
   - 气垫跑鞋
   - Nike Air Zoom
   ```

   **添加关键词**：
   - 输入关键词后按回车或点击"添加"
   - 点击关键词后的"×"可删除
   - 建议5-20个关键词

   **查看关键词搜索量**（新增功能）：
   - 添加关键词后，点击"📊 查看搜索量数据"按钮
   - 系统自动查询Google Ads Keyword Planner
   - 显示每个关键词的：
     * 月平均搜索量
     * 竞争程度（LOW/MEDIUM/HIGH）
     * 建议出价范围
   - 数据缓存7天，过期后需重新查询

   ![关键词搜索量数据](./images/keyword-metrics.png)

**可选字段**：

10. **行业类别**（帮助AI更好地生成创意）
    ```
    选项：运动用品、服装、电子产品、教育培训、餐饮美食等
    ```

![创建Offer表单](./images/create-offer.png)

#### Step 4: 保存Offer

点击页面底部的 **"保存Offer"** 按钮

**系统自动操作**：
- ✅ 自动分类关键词：
  - 品牌词 → EXACT匹配（精确匹配）
  - 产品词（2-3个词）→ PHRASE匹配（短语匹配）
  - 长尾词（≥4个词）→ BROAD匹配（广泛匹配）
- ✅ 保存到本地IndexedDB
- ✅ 跳转到Offer详情页

### 2.3 编辑和管理Offer

**查看Offer列表**：
- 显示所有已创建的Offers
- 可按状态筛选（活跃/暂停/已归档）
- 可搜索Offer名称或品牌

**编辑Offer**：
1. 点击Offer卡片
2. 进入详情页
3. 点击 **"编辑"** 按钮
4. 修改信息后保存

**删除Offer**：
1. 进入Offer详情页
2. 点击 **"删除Offer"**
3. 确认删除（⚠️ 此操作不可恢复）

## 三、AI生成广告创意

### 3.1 什么是广告创意？

广告创意包含：
- **15个Headlines**（标题，每个≤30字符）
- **4个Descriptions**（描述，每个≤90字符）
- **4个Callouts**（卖点，每个≤25字符）
- **4个Sitelinks**（附加链接，每个≤25字符）

这些内容会组合成Google响应式搜索广告（RSA）。

### 3.2 生成创意

#### Step 1: 进入创意生成页面

1. 在Offer列表点击某个Offer
2. 点击 **"生成广告创意"** 按钮

#### Step 2: AI自动生成（3级Fallback保障）

系统会自动尝试3种方式生成创意：

**Level 1: 网站分析**（优先）
```
正在分析您的官网...
- 提取网站标题、描述
- 提取导航菜单
- 提取独特卖点(USPs)
- 提取页面内容
```

如果网站分析失败（网站无法访问、反爬虫等），自动进入：

**Level 2: AI模板生成**（备选）
```
正在使用AI模板生成...
- 基于您填写的品牌名、产品描述
- 基于行业类别生成通用模板
- 基于目标关键词生成相关内容
```

如果AI生成失败（API错误、超时等），自动进入：

**Level 3: 手动输入**（保底）
```
自动生成失败，请手动输入创意内容
```

#### Step 3: 查看生成结果

**质量评分**（0-100分）：
- 90-100分：优秀 ✅
- 70-89分：良好 ✅
- 60-69分：及格 ⚠️
- <60分：建议重新生成 ❌

**评分维度**：
1. 关键词相关性（30%）：Headlines和关键词的匹配度
2. 创意多样性（20%）：表达方式的多样性
3. 关键词覆盖率（30%）：是否覆盖了所有目标关键词
4. 官网一致性（20%）：与官网内容的一致性

![创意生成结果](./images/creative-generated.png)

#### Step 4: 编辑创意内容（可选）

**Inline编辑**：
1. 点击任意Headlines或Descriptions
2. 直接编辑文本
3. 系统自动显示字符计数（如：25/30）
4. 超过字符限制会变红提示
5. 点击"✓"保存，点击"×"取消

**批量操作**：
- 选中多个Headlines → 批量删除
- 全部重新生成 → 点击"重新生成"按钮

#### Step 5: 保存创意

- 编辑完成后自动保存
- 系统保存完整版本历史
- 可随时查看和恢复历史版本

### 3.3 版本管理

**查看版本历史**：
1. 点击 **"版本历史"** 按钮
2. 显示所有生成和编辑的版本
3. 每个版本显示：
   - 时间戳
   - 质量评分
   - 来源（AI生成 / 手动编辑）

**对比版本**：
1. 选择2个版本
2. 点击 **"对比"**
3. 并排显示差异（红色=删除，绿色=新增）

**恢复历史版本**：
1. 选择要恢复的版本
2. 点击 **"恢复此版本"**
3. 确认恢复

## 四、一键上线广告

### 4.1 准备工作

**确认检查清单**：
- ✅ Google Ads账号已连接
- ✅ Offer信息完整
- ✅ 广告创意已生成（质量评分≥60分）
- ✅ 预算充足（建议每个广告系列≥¥100/天）

### 4.2 上线广告

#### Step 1: 点击"一键上广告"

在创意生成页面，点击页面右上角的 **"一键上广告"** 按钮

#### Step 2: 配置广告参数

**弹窗显示**：

1. **选择广告变体**（1-3个）

   **品牌导向**（必选）：
   - ✅ 强调品牌知名度、历史、口碑
   - 示例Headlines："Nike官方旗舰店"、"25年运动装备专家"

   **功能导向**（可选）：
   - 强调产品功能、技术参数
   - 示例Headlines："Air Zoom气垫缓震"、"透气网面设计"

   **价值导向**（可选）：
   - 强调用户利益、优惠、ROI
   - 示例Headlines:"全场8折优惠"、"免费配送30天退换"

   ![选择广告变体](./images/select-variants.png)

2. **确认预算**
   - 每个变体：¥100/天（固定）
   - 选择2个变体 = 总预算¥200/天
   - 选择3个变体 = 总预算¥300/天

3. **成本预估**（基于Google Ads Keyword Planner）
   - 预估平均CPC：¥1.50
   - 预估每日点击量：67次
   - 预估每日展示量：1340次
   - 预估每月花费：¥3000

4. **高价值关键词推荐**（如果有历史数据）
   ```
   💡 基于您之前的Nike品牌广告系列，推荐以下高价值关键词：
   - Nike Air Zoom（CTR: 8.5%, CPC: ¥1.20）
   - 专业跑鞋（CTR: 7.2%, CPC: ¥1.35）
   - 马拉松跑鞋（CTR: 6.8%, CPC: ¥1.45）

   [✓] 自动添加这些关键词到新广告系列
   ```

#### Step 3: 确认并上线

点击 **"确认上线"** 按钮

**系统自动执行10步流程**：

```
进度条显示：

✓ Step 1/10: 创建广告预算... (2秒)
✓ Step 2/10: 创建广告系列... (3秒)
✓ Step 3/10: 创建广告组... (2秒)
✓ Step 4/10: 创建响应式搜索广告... (4秒)
✓ Step 5/10: 添加关键词... (3秒)
✓ Step 6/10: 添加Sitelinks... (3秒)
✓ Step 7/10: 添加Callouts... (2秒)
✓ Step 8/10: 设置地理位置定位... (2秒)
✓ Step 9/10: 设置广告时间... (2秒)
✓ Step 10/10: 启用广告系列... (2秒)

🎉 广告上线成功！
```

**总耗时**：约25-30秒

![上线进度](./images/campaign-progress.png)

#### Step 4: 上线成功

**成功页面显示**：
- ✅ 广告系列创建成功
- 📊 Campaign ID: 1234567890
- 🔗 Google Ads控制台链接

**重要提示**：
```
⚠️ 请在Google Ads控制台完成以下手动操作：

1. 上传Images（图片）：
   - 访问Google Ads控制台
   - 进入广告系列设置
   - 点击"Add images to your campaign"
   - 上传至少3张图片（建议尺寸：1200×628）

2. 上传Business Logo（商标）：
   - 上传品牌Logo（建议尺寸：1200×1200）
   - 支持PNG或JPG格式

📖 详细教程：[点击查看图片上传指南]
```

### 4.3 验证广告

**在Google Ads控制台验证**：
1. 点击"访问Google Ads控制台"链接
2. 在Campaign列表找到新创建的广告系列
3. 检查状态：应为"已启用"或"正在审核"
4. 检查Ad：应显示您的Headlines和Descriptions

**Google广告审核**：
- 审核时间：1-2个工作日
- 审核通过后广告开始投放
- 如有违规，Google会发邮件通知

### 6.4 一键调整CPC（新增功能）

**功能说明**：
快速调整某个Offer关联的所有广告系列的CPC出价，无需逐个进入Google Ads控制台修改。

**使用场景**：
- 广告表现良好，想提高出价增加曝光
- 成本过高，想降低CPC节省预算
- 统一调整所有相关广告的出价策略

#### 如何调整CPC

**Step 1: 找到要调整的Offer**
1. 进入Offers列表页
2. 找到目标Offer卡片
3. 点击卡片上的 **"💰 调整CPC"** 按钮

**Step 2: 选择调整方式**
```
弹窗显示3种调整方式：

○ 提高CPC：按百分比提高出价
  示例：提高15% → ¥5.00变为¥5.75

○ 降低CPC：按百分比降低出价
  示例：降低20% → ¥5.00变为¥4.00

● 设为固定值：设置为指定金额
  示例：设为¥6.50 → 所有广告CPC都改为¥6.50
```

**Step 3: 输入调整幅度或数值**
```
输入框：[15] %/元

提示：
- 提高/降低模式：输入百分比（建议不超过±50%）
- 固定值模式：输入具体金额（范围：¥0.10 - ¥100.00）
```

**Step 4: 预览调整结果**
```
系统自动显示预览表格：
```

```
✅ CPC调整成功

失败：0个

调整立即生效，无需等待审核
```

#### 注意事项

**安全限制**：
- ⚠️ 单次调整幅度不超过±50%
- ⚠️ CPC范围限制：¥0.10 - ¥100.00
- ⚠️ 单日最多调整3次，避免频繁操作
- ⚠️ 不同竞价策略的调整行为可能不同

**竞价策略差异**：

   - 调整CPC上限
   - 系统在此上限内自动优化

3. **Maximize Conversions（尽可能提高转化次数）**：
   - 调整目标CPA
   - 系统自动调整出价以达到目标

**最佳实践**：
- 📊 查看近7天表现后再决定调整方向
- 📈 表现好的Campaign提高10-20%出价
- 📉 表现差的Campaign降低20-30%出价或暂停
- ⏰ 避免频繁调整，建议至少间隔7天
- 📝 记录每次调整和后续效果，总结经验

#### 如何解读分析结果？

**1. 总评分（0-100分）**

总分是5个维度的加权总和：
- **80-100分**：优秀，强烈建议投放
- **65-79分**：良好，建议投放但需优化
- **50-64分**：中等，建议优化后投放
- **35-49分**：较差，需大幅优化
- **0-34分**：不建议投放

**2. 各维度评分**

每个维度显示：
- 得分/满分（如：24/30）
- 百分比（如：80%）
- 进度条可视化
- 详细指标数据

**重点关注**：
- ⚠️ 低于60%的维度是主要短板，优先优化
- ✅ 高于80%的维度是核心优势，可以强化

**3. AI综合评价**

AI基于所有维度综合分析，给出：
- 整体投放建议
- 市场竞争评估
- 投放策略方向

**4. 优势与劣势**

**主要优势（✅）**：
- 列出Offer的核心竞争力
- 这些是您应该在广告中突出的卖点

**需要改进（⚠️）**：
- 列出明显的短板和风险
- 这些是投放前应该优化的方面

**5. AI优化建议**

每条建议包含：
- **优先级标签**：[高优先级]/[中优先级]/[低优先级]
- **类型标签**：关键词/预算/着陆页/定向/内容
- **具体建议**：可操作的优化方向
- **预期影响**：量化的改进效果预测

**建议的优先级含义**：
- **高优先级**：对投放效果影响显著，建议立即实施
- **中优先级**：有一定改进空间，建议尽快实施
- **低优先级**：锦上添花的优化，可按需实施

**6. ROI预测**

基于历史数据和市场基准，AI预测3种场景：

**保守预测（70%概率）**：
- 最保守的估计，达成概率最高
- 适合用于预算规划和风险评估

**适中预测（50%概率）**：
- 中性场景下的表现
- 适合用于目标设定

**乐观预测（20%概率）**：
- 最理想的表现
- 适合用于潜力评估

**ROI计算公式**：
```
ROI = (预计收入 - 广告支出) / 广告支出 × 100%
```

**如何使用ROI预测**：
- 如果保守预测的ROI >50%，投放风险较低
- 如果适中预测的ROI >100%，投放性价比较高
- 如果乐观预测的ROI >200%，投放潜力巨大

## 七、常见问题FAQ

### Q1: 广告创建失败，提示"Token已过期"

**原因**：Google OAuth Token有效期通常为1小时

**解决方案**：
1. 点击顶部的 **"重新授权"** 按钮
2. 重新完成OAuth授权流程
3. 重试创建广告

### Q2: AI生成的创意质量评分很低（<60分）

**原因**：
- 网站信息不足
- 产品描述过于简单
- 关键词与产品不相关

**解决方案**：
1. 完善Offer的产品描述（至少100字）
2. 添加更多相关关键词（至少10个）
3. 检查落地页URL是否可访问
4. 手动编辑创意内容
5. 点击 **"重新生成"**

### Q3: Dashboard没有显示数据

**可能原因**：
1. 广告刚上线，还没有数据（需要等待几小时）
2. 数据未同步
3. 选择的日期范围内无数据

**解决方案**：
1. 点击 **"刷新数据"** 手动同步
2. 调整日期范围为"近30天"
3. 检查广告在Google Ads控制台的状态

### Q4: 广告一直显示"正在审核"

**原因**：Google Ads需要审核广告内容

**正常审核时间**：
- 通常：1-2个工作日
- 最长：7个工作日

**加快审核**：
- 确保广告内容符合Google政策
- 避免敏感词汇
- 落地页可访问且内容相关

**如果审核不通过**：
- Google会发邮件通知不通过原因
- 修改广告内容后重新提交

### Q5: 我可以在多个设备上使用AutoAds吗？

**回答**：可以，但需要数据导出/导入

**步骤**：
1. 设备A：导出数据
2. 设备B：导入数据
3. 设备B：重新连接Google Ads账号（Token不可跨设备）

**限制**：
- 数据存储在浏览器本地，不同步到云端
- 每个设备需要独立授权Google Ads

### Q6: 清除浏览器数据后，我的Offers和数据都没了！

**原因**：数据存储在浏览器IndexedDB，清除数据会删除

**预防措施**：
1. 定期导出数据备份（建议每周）
2. 重要Offer创建后立即导出
3. 不要随意清除浏览器数据

**恢复**：
- 如果有备份：导入之前导出的JSON文件
- 如果无备份：需要重新创建Offers（Google Ads上的广告仍在运行）

### Q7: 我可以管理100个Google Ads账号吗？

**回答**：可以，但建议分批管理

**建议**：
- 单次连接≤20个账号（避免浏览器卡顿）
- 数据量大时可能影响性能
- 可创建多个"工作区"分别管理不同账号组

### Q8: 高价值关键词会自动应用到所有新广告吗？

**回答**：不会，需要手动确认

**工作流程**：
1. 系统在创建新Offer时**提示**有可用的高价值关键词
2. 用户选择是否添加
3. 可以全部添加、手动选择或跳过

**原因**：
- 不同Campaign目标可能不同
- 用户应保留最终控制权
