## 🟡 重要问题 (P1) - 影响核心用户体验

### P1-1: Offer创建流程缺少字段
**需求**: 需求1和28要求Offer包含以下字段：
- 推广链接 ✅
- 品牌名称 ✅
- 推广国家 ✅
- 店铺/商品落地页 ✅
- product_price (可选) ❓
- commission_payout (可选) ❓
**现状**: 需验证product_price和commission_payout字段是否在创建表单中
**影响**: 无法计算"建议最大CPC"
**修复**:
- 在Offer创建/编辑表单添加"产品价格"和"佣金比例"字段
- 标注为"可选"
- 添加示例值（如 $699.00, 6.75%）

---

### P1-2: 广告创意生成后缺少评分功能
**需求**: 需求17要求"在广告创意生成后，需要对生成的广告质量进行评分（满分100分），并支持点击'重新生成'按钮多次尝试"
**现状**: 需验证LaunchAdModal中是否有评分和重新生成功能
**影响**: 无法保证广告质量
**修复**:
- 广告创意生成后显示评分（0-100分）
- 评分旁边显示"重新生成"按钮
- 评分低于70分时给予警告提示
- 用户满意后才能继续后续流程

---

### P1-3: 缺少广告变体差异化设置
**需求**: 需求16要求"用户可选择一次上线1-3个有差异的广告，确保广告创意、关键词、ads广告等都是差异化的，且广告变体中一定要包含'品牌导向'"
**现状**: 需验证是否支持多变体创建
**影响**: 无法快速A/B测试
**修复**:
- 在"一键上广告"流程中添加"广告数量"选择（1-3个）
- 每个变体生成不同的创意和关键词
- 至少一个变体必须是"品牌导向"
- 清晰标注每个变体的特点

---

### P1-4: 默认广告参数可能不可修改
**需求**: 需求14要求"一键上广告功能的部分选项的默认值，支持用户手动修改"，包括：
- objective (默认: Website traffic)
- Conversion goals (默认: Page views)
- campaign type (默认: Search)
- bidding strategy (默认: Maximize clicks)
- Maximum CPC bid limit (默认: CN¥1.2 或 US$0.17)
- budget (默认: 100 对应货币单位)
- EU political ads (默认: No)
**现状**: 需验证这些字段是否都可以在弹窗中修改
**影响**: 用户无法根据实际情况调整参数
**修复**:
- 确保所有默认参数在UI中可见且可编辑
- 使用下拉框/输入框形式
- 显示推荐值，但允许用户修改

---

### P1-5: CPC建议计算公式可能未实现
**需求**: 需求28要求"配置CPC处文案提醒'建议最大CPC'：最大CPC = product_price * commission_payout / 50，且需要汇率转换"
**现状**: 需验证是否有自动计算和显示
**影响**: 用户不知道合理的CPC范围
**修复**:
- 在CPC配置区域显示"建议最大CPC"
- 根据product_price和commission_payout自动计算
- 考虑账号货币单位进行汇率转换
- 显示计算公式说明

---

## 🎯 UX用户体验问题分析

### 核心用户流程评估

#### 流程1: 创建Offer → 上广告
**期望流程** (根据需求):
1. 登录
2. 创建Offer（填写推广链接、品牌、国家、落地页）
3. Offer列表看到新Offer
4. 点击"投放评分"查看评分
5. 点击"一键上广告"
6. 选择广告数量（1-3个）
7. 查看AI生成的创意和评分
8. 满意后确认配置（budget, CPC等）
9. 广告上线

**当前问题**:
- ❌ 缺少"投放评分"明确入口
- ❌ 广告创意评分功能不确定
- ❌ 多变体支持不确定

#### 流程2: 查看数据 → 优化广告
**期望流程**:
1. 访问Dashboard
2. 查看KPI总览
3. 查看活跃广告系列表现
4. 查看风险提示
5. 点击表现差的广告系列
6. 调整CPC或暂停
7. 查看优化后效果

**当前问题**:
- ❌ 缺少"风险提示"板块
- ⚠️ 需验证是否可直接从Dashboard跳转到广告详情
- ⚠️ 需验证CPC调整流程

#### 流程3: 批量管理Offer
**期望流程**:
1. 访问Offer列表
2. 点击"批量导入"
3. 下载模板
4. 填写Offer信息
5. 上传CSV
6. 预览导入数据
7. 确认导入
8. 查看导入结果

**当前问题**:
- ⚠️ 需验证模板下载功能
- ⚠️ 需验证导入预览功能
- ⚠️ 需验证错误处理

### 响应时间
| 操作 | 期望 | 需验证 |
|------|------|--------|
| 页面加载 | < 2秒 | ✓ |
| Offer创建 | < 3秒 | ✓ |
| 广告创意生成 | < 10秒 | ✓ |
| 数据导出 | < 5秒 | ✓ |

### 错误处理
| 场景 | 期望 | 需验证 |
|------|------|--------|
| 网络错误 | 友好错误提示 + 重试按钮 | ✓ |
| 表单验证失败 | 实时高亮错误字段 | ✓ |
| API错误 | 具体错误信息（非技术术语） | ✓ |
| 权限不足 | 清晰说明原因 + 联系管理员 | ✓ |
